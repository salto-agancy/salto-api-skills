# Яндекс.Бизнес — метрики карточки организации

Карточка организации в Я.Бизнес = отдельный **счётчик Метрики**, прицепленный к этой карточке.

## Как найти counter_id карточки

1. Я.Бизнес → Ваша организация → **Статистика** → Подробная статистика
2. Внизу страницы — ссылка на Метрику
3. Или в URL статистики Метрики: `https://metrika.yandex.ru/dashboard?id=XXXXXXXX`

⚠️ Этот счётчик ≠ счётчик вашего сайта. Они разные.

## Что считается "визитом" в карточке

- Открытие карточки на Картах/Поиске/в приложении
- Клик "Подробнее", "Фото", "Отзывы" — это переходы внутри карточки

## Ключевые цели (goals) карточки

У Яндекс.Бизнес автоматически созданы цели:
- **Клик по телефону** (`CallClick`)
- **Проложить маршрут** (`BuildRoute`)
- **Переход на сайт** (`SiteClick`)
- **Фото организации** (`PhotoOpen`)
- **Отзывы** (`ReviewsOpen`)

Список твоих целей → `scripts/goals_report.py <counter_id>`.

## Типичные метрики отчёта по карточке

| Метрика | Как получить |
|---------|--------------|
| Переходы в профиль | `ym:s:visits` |
| Уникальные пользователи | `ym:s:users` |
| Клики по телефону | `ym:s:goal<ID_CallClick>reaches` |
| Маршруты | `ym:s:goal<ID_BuildRoute>reaches` |
| Переходы на сайт | `ym:s:goal<ID_SiteClick>reaches` |

## Разбивка по рекламному/органическому трафику

В Я.Бизнес можно платить за продвижение карточки (это через платную подписку). Тогда часть визитов будет `trafficSource=='ad'`.

```
filters=ym:s:trafficSource=='ad'
```
— отделит платный охват карточки от органического.

## Ограничения

- Метрика карточки **не показывает показы на Картах** — только **взаимодействия** (переходы в профиль, клики). Показы видны только в кабинете Я.Бизнес.
- Для показов (Impressions на Картах) используй встроенную статистику Я.Бизнес в admin-панели (нет публичного API по ним).

## Пример запроса

```python
# scripts/business_card.py
from query import stat_query
from goals_report import list_goals

counter = '97281982'  # счётчик карточки
goals = list_goals(counter, token)

# Достижения каждой цели за месяц
for g in goals:
    r = stat_query(counter, f'ym:s:goal{g["id"]}reaches',
                   date1='2026-03-01', date2='2026-03-31', token=token)
    print(f'{g["name"]}: {r["totals"][0]:.0f}')
```

## Типичный отчёт

```

Проложено маршрутов: 89
Клики по телефону: 68
Переходы на сайт: 58
```
